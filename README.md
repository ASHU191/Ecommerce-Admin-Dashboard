# E-commerce app with Admin Dashboard using Node & React Redux-toolkit 

## Live Demo                                                                                                         
   
[View live demo]     
                                                                                                                                                                 
## installation      
              
 
```                                                             
    npm install
```                                                                        

## Commands for run the app
                                   
                                                                                                                                                        
                                
```                                                                                                                                  
                                                                                                                                  
                                               
    npx json-server --watch src/data/db.json --port 8000
```   
```                                      
    npm start 
```

- Node JS

- Redux

- React-router-dom v6
- React-paginate
- Json-server
- React-toastify
- Swiper
- Bootstrap
